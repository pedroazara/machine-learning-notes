# Machine Learning – Syllabus

## Objectives
Understand the theoretical foundations and practical implementation of classical machine learning algorithms.

## Topics

1. Introduction to Machine Learning
   - Supervised vs unsupervised learning
   - Train/validation/test split

2. Linear Regression
   - Least squares
   - Gradient descent

3. Logistic Regression
   - Maximum likelihood
   - Decision boundaries

4. K Nearest Points (KNN)

5. Bias–Variance Tradeoff

6. Regularization
   - L1 and L2 penalties

7. Support Vector Machines
   - Maximum margin
   - Kernel trick

8. Tree-based Models
   - Decision trees
   - Random forests

9. Model Evaluation
   - Confusion matrix
   - ROC and AUC
   - F1-score

10. Cross-validation and hyperparameter tuning

## Tools
Python, NumPy, Pandas, scikit-learn, Matplotlib
